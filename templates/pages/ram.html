Random Access Memory

While our favorite application is running it consumes a memory to save short term information. That is the function of RAM, it was used by the softwares that you used to store short term information then delete it after the application was used. RAM also helps our computer to run faster. The higher ram , the more application that you can open simultaneously.

Be aware of buying RAM, because it differs depending on you processor and motherboard compatibility. Because there different types of RAM. These are DDR1 upto DDR4 RAM. Also, keep in practice that whenever we buy ram make sure that its the same brand and have the same gb and mhz to avoid having a problem. Though using different brands and different sizes will still work but the problem in my experience it will produce some glitching and graphics problem.
